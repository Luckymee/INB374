Order

The ordering service is a WSDL based web service that interacts with the orders, order details and products database tables allowing the sales team to create orders. The service can be found here, its associated WSDL file here.

Functionality

addOrder

Places an order in database.

addOrderDetails

Places within the database the individual details of an order as well as manage any stock level changes that may occur with an order.

updateExistingOrders

Iterates through the orders and updates the status of any that have since been shipped to the customer.
