Supplier

The supplier service is a WSDL based web service that allows the warehouse to interact with the supplier. The service can be found here, its associated WSDL file here.

Functionality

getWaitTime

Returns the delivery time of stock to the warehouse.

restockWarehouse

Compiles a list of all stock that the warehouse wishes to order.
